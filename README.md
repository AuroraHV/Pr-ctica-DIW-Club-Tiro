# CdTiro-CostaAlmería

Proyecto web estático para la asignatura DIW, centrado en la presentación del Club Deportivo de Tiro Costa de Almería.

Incluye varias páginas HTML (inicio, producto, búsqueda, carrito y contacto), estilos personalizados en CSS y lógica en JavaScript para interacciones como modo oscuro, carrito dinámico y componentes interactivos.
